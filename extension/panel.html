ConnectIQ
ConnectIQ, allows the user to proactively improve their linkdIn presence by providing AI assitance to post, comment and boost their professional connections.


As you install this extension, you will find the following options inside your linkdIn page.
  • You will find the a custom input bar where you can provide a prompt on how the comment should look like and on submitting, a comment will created for the specific post.
  • You also have an option to select any predefined tone for your comment by clicking pencil icon right beside the input bar.
